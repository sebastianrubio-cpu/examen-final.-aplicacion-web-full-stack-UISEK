# examen-final.-aplicaci-n-web-full-stack-UISEK
Proyecto desarrollado como examen final de Desarrollo Web en la UISEK. Consiste en una aplicación web Full-Stack que integra frontend, backend y base de datos, implementando autenticación, operaciones CRUD y una interfaz responsiva, aplicando buenas prácticas de desarrollo, seguridad y arquitectura de software.
